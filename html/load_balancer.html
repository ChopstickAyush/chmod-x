class load_balancer.LoadBalancerCPUUtil(n_ports, ports, cursor)[source]¶

Bases: object

This class is designed for load balancing strategy according to the load at cpu which is monitored according to the memory allocated. The least occupied server is given the duty of message. The record is maintained in a database.
