client module¶

class client.GUI(master, ports)[source]¶

Bases: object

This is the implementation of the client side GUI, that interacts with the server.

Parameters:

  • current_client_socket – This holds the socket object initialized with TCP anf IPv4

  • root (Tk) – This is the main Tkinter window

  • chat_transcript_area (Text) – This is the box in the GUI where the chat appears

  • has_joined (bool) – This is a variable indicating whether the client has connected to the server.

  • has_registered (bool) – This is a variable indicating whether the client has been registered in the server side database

  • current_group (string) – This indicates the current group the user is in

  • name_widget (Entry) – This is the Tkinter widget that takes the username input

  • pass_widget (Entry) – This is the Tkinter widget that takes the password input

  • enter_text_widget (Entry) – This is the Tkinter widget that takes the chat message from the user

  • join_button (Button) – On pressing this button the user gets connected to the server

  • create_group_button (Button) – This allows the client to create/amend any group(if it exists)

  • join_group_button (Button) – This allows the client to join any group they it is already a member

  • sign_up_button (Button) – This allows the client to resgiter their username into the server

  • group_name_widget (Entry) – This is where the group name is inputted while creating a group

  • add_members_widget (Entry) – This is the comma separted values of the members who are to be added into the group

  • join_group_name_widget (Entry) – This is where the group name is inputted while joining a group
